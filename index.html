AvocadoBot

AvocadoBot is a simple Discord chat bot for streaming music and sharing both GIFs & images directly to your guild.

Technology

The Discord bot is built with Java 8 and Gradle. The YouTube API is leveraged for supporting real-time music streaming, user searches, and playlist management. Integration with the Imgur allows for sharing GIFs and images directly to Discord chat channels. Hosted on an Ubuntu AWS EC2 instance, the bot is usable through the Discord desktop and web apps.

ARChess

ARChess is an IoT device and web application created for a graduate level Human Augmentics research class, with a team of 7 engineers. The aim of the device is to improve skill development in chess using augmented reality.

Technology

A custom chess board was designed to capture the board state of the game. That information is fed to an Arduino, which sends the data to the web app. The Stockfish.js chess engine then calculates the player's next best move, and visualizes the strategic information in real-time via an augmented reality webcam overlay built with JavaScript, Awe.js, and dat.GUI.

JavaScript Node.js Awe.js
